Repo Default Config

Yocto Repo

  • 创建自己layer仓库；

  • 参考下面fsl-setup-release.sh添加layer到conf/bblayers.conf方式，换成自己的fsl-setup-release.sh；

  • 修改下面的default.xml成自己的需求，push到github仓库里；

  • repo init -u <your github repository url> -b <your github repository branch>；
