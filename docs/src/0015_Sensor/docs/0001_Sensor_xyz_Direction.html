  • kernel-4.9/drivers/misc/mediatek/sensors-1.0/hwmon/hwmsen/hwmsen_helper.c

    struct hwmsen_convert map[] = {
    { { 1, 1, 1}, {0, 1, 2} },
    { {-1, 1, 1}, {1, 0, 2} },
    { {-1, -1, 1}, {0, 1, 2} },
    { { 1, -1, 1}, {1, 0, 2} },

    { {-1, 1, -1}, {0, 1, 2} },
    { { 1, 1, -1}, {1, 0, 2} },
    { { 1, -1, -1}, {0, 1, 2} },
    { {-1, -1, -1}, {1, 0, 2} },

};

  • 如上map为:

    • xyz正负方向变换

    • xyz轴位置重映射
