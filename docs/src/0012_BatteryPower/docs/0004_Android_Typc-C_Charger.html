简述

MTK方案的电池充电过程分为预充、恒流充电（CC模式）、恒压充电（CV模式）三种模式，从充电状态图看出来，刚开始充电的时候，代码先判断是插USB充电还是插ac充电，电池在进入充电阶段分为快速充电、CC（恒流充电）、CV（恒压充电）。

Charger所处的位置，在通往电池Battery的那条通路上，会有一个毫欧级别的电阻，两端接着Gauge，以此来判定电池电流状态
