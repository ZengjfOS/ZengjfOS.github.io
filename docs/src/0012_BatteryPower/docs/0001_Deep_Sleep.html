STR(Suspend to RAM): жҢӮиө·еҲ°еҶ…еӯҳпјҢдҝ—з§°еҫ…жңәгҖҒзқЎзң пјҲSleepпјүпјҢиҝӣе…ҘиҜҘзҠ¶жҖҒпјҢзі»з»ҹзҡ„дё»иҰҒе·ҘдҪңеҰӮдёӢпјҡ

  1. е°Ҷзі»з»ҹеҪ“еүҚзҡ„иҝҗиЎҢзҠ¶жҖҒзӯүж•°жҚ®дҝқеӯҳеңЁеҶ…еӯҳдёӯпјҢжӯӨж—¶д»ҚйңҖиҰҒеҗ‘RAMдҫӣз”өпјҢд»ҘдҝқиҜҒеҗҺз»ӯеҝ«йҖҹжҒўеӨҚиҮіе·ҘдҪңзҠ¶жҖҒ

  2. еҶ»з»“з”ЁжҲ·жҖҒзҡ„иҝӣзЁӢе’ҢеҶ…ж ёжҖҒзҡ„д»»еҠЎпјҲиҝӣе…ҘеҶ…ж ёжҖҒзҡ„иҝӣзЁӢжҲ–еҶ…ж ёиҮӘе·ұзҡ„taskпјү

  3. е…ій—ӯеӨ–еӣҙи®ҫеӨҮпјҢеҰӮжҳҫзӨәеұҸгҖҒйј ж Үзӯү,дёӯж–ӯе”ӨйҶ’еӨ–и®ҫдёҚдјҡе…ій—ӯпјҢеҰӮз”өжәҗй”®

  4. CPUеҒңжӯўе·ҘдҪң

Standbyд№ҹеұһдәҺзқЎзң зҡ„дёҖз§Қж–№ејҸпјҢеұһдәҺжө…зқЎзң гҖӮиҜҘжЁЎејҸдёӢCPUе№¶жңӘж–ӯз”өпјҢдҫқж—§еҸҜд»ҘжҺҘж”¶еӨ„зҗҶжҹҗдәӣзү№е®ҡдәӢд»¶пјҢи§Ҷе…·дҪ“и®ҫеӨҮиҖҢе®ҡпјҢжҒўеӨҚиҮіжӯЈеёёе·ҘдҪңзҠ¶жҖҒзҡ„йҖҹеәҰд№ҹжҜ”STRжӣҙеҝ«пјҢдҪҶд№ҹжӣҙдёәиҖ—з”өгҖӮдёҫдёӘдҫӢеӯҗжқҘиҜҙпјҢд»ҘиҜҘж–№ејҸиҝӣе…ҘзқЎзң ж—¶пјҢеҗҺз»ӯйҖҡиҝҮзӮ№еҮ»й”®зӣҳд№ҹиғҪе°Ҷзі»з»ҹе”ӨйҶ’гҖӮиҖҢд»Ҙmemиҝӣе…Ҙзҡ„зқЎзң дёәж·ұеәҰзқЎзң пјҢеҸӘиғҪйҖҡиҝҮдёӯж–ӯе”ӨйҶ’и®ҫеӨҮе”ӨйҶ’зі»з»ҹпјҢеҰӮз”өжәҗй”®(жӯӨж—¶жҢүз”өжәҗй”®пјҢдёҚдјҡз»ҸиҝҮжӯЈеёёзҡ„ејҖжңәжөҒзЁӢзҡ„BIOSгҖҒBOOTLOADзӯү)пјҢжӯӨж—¶жҢүй”®зӣҳжҳҜж— жі•е”ӨйҶ’зі»з»ҹзҡ„гҖӮ

STD(Suspend to Disk): жҢӮиө·еҲ°зЎ¬зӣҳпјҢдҝ—з§°дј‘зң пјҲHibernationпјүе°Ҷзі»з»ҹеҪ“еүҚзҡ„иҝҗиЎҢзҠ¶жҖҒзӯүж•°жҚ®дҝқеӯҳеҲ°зЎ¬зӣҳдёҠпјҢе№¶иҮӘеҠЁе…іжңәгҖӮдёӢж¬ЎејҖжңәж—¶дҫҝд»ҺзЎ¬зӣҳдёҠиҜ»еҸ–д№ӢеүҚдҝқеӯҳзҡ„ж•°жҚ®пјҢжҒўеӨҚеҲ°дј‘зң е…іжңәд№ӢеүҚзҡ„зҠ¶жҖҒгҖӮ иӯ¬еҰӮеңЁдј‘зң е…іжңәж—¶пјҢжЎҢйқўжү“ејҖдәҶдёҖдёӘеә”з”ЁпјҢйӮЈд№ҲдёӢдёҖж¬ЎејҖжңәеҗҜеҠЁж—¶пјҢиҜҘеә”з”Ёд№ҹеӨ„дәҺжү“ејҖзҠ¶жҖҒгҖӮиҖҢжӯЈеёёзҡ„е…іжңә-ејҖжңәжөҒзЁӢпјҢиҜҘеә”з”ЁжҳҜдёҚдјҡжү“ејҖзҡ„гҖӮ

еңЁж–°зүҲеҶ…ж ёдёӯпјҢиҝӣзЁӢfreezeзҡ„еҠҹиғҪиў«еҚ•зӢ¬жҠҪзҰ»еҮәжқҘдҪңдёәдёҖдёӘз”өжәҗзҠ¶жҖҒпјҢиҜҘзҠ¶жҖҒд»…д»…жҳҜеҶ»з»“иҝӣзЁӢпјҢе№¶дёҚдјҡдҪҝзі»з»ҹиҝӣе…ҘдҪҺеҠҹиҖ—зҠ¶жҖҒпјҲеҰӮеҲҮж–ӯCPUж—¶й’ҹжәҗгҖҒе…ій—ӯеӨ–и®ҫдҫӣз”өзӯүпјүгҖӮзӣёеҪ“дәҺжҳҜStandbyзҠ¶жҖҒгҖӮ

autosleepд№ҹжҳҜд»ҺAndroid wakelocksиЎҘдёҒйӣҶдёӯжј”еҢ–иҖҢжқҘзҡ„пјҲLinuxз”өжәҗз®ЎзҗҶ(9)_wakelocksпјүпјҢз”ЁдәҺеҸ–д»ЈAndroid wakelocksдёӯзҡ„иҮӘеҠЁдј‘зң еҠҹиғҪгҖӮе®ғеҹәдәҺwakeup sourceе®һзҺ°гҖӮautosleepзҡ„еҠҹиғҪеҫҲе·Із»ҸеҫҲзӣҙзҷҪдәҶпјҢвҖқзі»з»ҹжІЎжңүдәӢжғ…еңЁеҒҡвҖқзҡ„ж—¶еҖҷпјҢе°ұе°Ҷзі»з»ҹеҲҮжҚўеҲ°дҪҺеҠҹиҖ—зҠ¶жҖҒгҖӮautosleepзҡ„е®һзҺ°дҪҚдәҺkernel/power/autosleep.cдёӯпјҢеҹәдәҺwakeup countе’Ңsuspend & hibernateеҠҹиғҪпјҢе№¶йҖҡиҝҮPM coreзҡ„mainжЁЎеқ—еҗ‘з”ЁжҲ·з©әй—ҙжҸҗдҫӣsysfsж–Үд»¶пјҲ/sys/power/autosleepпјүгҖӮ

зӣёжҜ”Android wakelocksпјҢKernel wakelocksзҡ„е®һзҺ°йқһеёёз®ҖеҚ•пјҲз®ҖеҚ•зҡ„жүҚжҳҜжңҖеҘҪзҡ„пјүпјҢе°ұжҳҜеңЁPM coreдёӯеўһеҠ дёҖдёӘwakelockжЁЎеқ—пјҲkernel/power/wakelock.cпјүпјҢиҜҘжЁЎеқ—дҫқиө–wakeup events frameworkжҸҗдҫӣзҡ„wakeup sourceжңәеҲ¶пјҢе®һзҺ°з”ЁжҲ·з©әй—ҙзҡ„wakeup sourceпјҲе°ұжҳҜwakelocksпјүпјҢе№¶йҖҡиҝҮPM core mainжЁЎеқ—пјҢеҗ‘з”ЁжҲ·з©әй—ҙжҸҗдҫӣдёӨдёӘеҗҢеҗҚзҡ„sysfsж–Үд»¶пјҢwake_lockе’Ңwake_unlockгҖӮ
