使用Android R版本的dnsmasq分析

Android Version

  • external/dnsmasq/README.version

    URL: http://www.thekelleys.org.uk/dnsmasq/dnsmasq-2.51.tar.gz
Version: 2.51
BugComponent: 31808

Download and Compile

  • wget http://www.thekelleys.org.uk/dnsmasq/archive/dnsmasq-2.51.tar.gz

    • tar xvf dnsmasq-2.51.tar.gz

    • cd dnsmasq-2.51

    • make

Run

  • sudo src/dnsmasq –keep-in-foreground –no-resolv –no-poll –dhcp-authoritative –pid-file -C dnsmasq.conf.example

    • –no-resolv: 表示没有resolv配置文件
