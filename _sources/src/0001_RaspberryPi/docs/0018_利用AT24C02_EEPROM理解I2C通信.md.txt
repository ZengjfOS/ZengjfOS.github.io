# 利用AT24C02 EEPROM理解I2C通信

利用Overlay加载AT24C02 I2C驱动，通过i2c-tools获取数据

# overlay

* arch/arm/boot/dts/overlays/rpi-at24c02-overlay.dts
  ```dts
  // rpi-sense HAT
  /dts-v1/;
  /plugin/;
  
  / {
          compatible = "brcm,bcm2835";
  
          fragment@0 {
                  target = <&i2c1>;
                  __overlay__ {
                          #address-cells = <1>;
                          #size-cells = <0>;
                          status = "okay";
  
                          DebugHAT@50 {
                                  compatible = "atmel,24c02";
                                  reg = <0x50>;
                                  status = "okay";
                          };
                  };
          };
  };
  ```

# 操作EEPROM

* sudo su
* ls -al /sys/bus/i2c/devices/1-0050
* echo zengjf > eeprom
* xxd -g 1 eeprom
