# README

用于分析USB CY7C68013A芯片通信

注意：
1. 按键这部分单片机没有任何上下拉，所以在按键外围需要自己加1k上拉，10k下拉，再加个10uF去抖，看情况自己调整就行了。
2. 左边是Master，右边是Slave
